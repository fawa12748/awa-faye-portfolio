## 📌 Présentation du projet

Ce projet est un **portfolio statique en ligne** développé dans le cadre de mon **cours de développement web** en **Licence 2 Géomatique** à l'**Université Iba Der Thiam de Thiès (UIDT)**.

L'objectif est de créer une **vitrine professionnelle** présentant :
- Mes compétences en géomatique et développement web
- Mes projets réalisés (SIG, analyse de données, développement web)
- Mes informations de contact
- Une galerie d'images de mes travaux

Ce portfolio permet aux recruteurs et collaborateurs de découvrir mon travail et de me contacter facilement.

## 🎯 Objectifs du projet

### Objectifs fonctionnels
- ✅ Page d'accueil avec présentation personnelle
- ✅ Galerie de projets (descriptions, technologies, liens)
- ✅ Formulaire de contact avec validation JavaScript
- ✅ Affichage des compétences avec niveaux
- ✅ Design responsive (mobile, tablette, desktop)
- ✅ Galerie d'images avec téléchargement
- ✅ Mode sombre / clair

### Objectifs pédagogiques
- ✅ Structurer une page web avec HTML sémantique
- ✅ Styliser l'interface avec CSS (Flexbox, Grid, animations)
- ✅ Manipuler le DOM en JavaScript
- ✅ Gérer les événements (clic, soumission de formulaire)
- ✅ Simuler l'envoi d'un formulaire
- ✅ Organiser le code en plusieurs fichiers
